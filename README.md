# Movie Rec App

Time spent: 4 hours 

## Features

The app includies the following functionalites:

- [X] User can scroll and browse through a list of movies, called by an API
- [X] User can click on specified movie and get info on:
    - [X] Title cover
    - [X] Summary
    - [X] Tating
    - [X] Release date
